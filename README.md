## Description

REST API implemented with [Nest.js](https://docs.nestjs.com/) and [Swagger](https://swagger.io/) as an example application for the [article]().

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
